# AIU AI Chatbot System

This project is an AI-powered conversational chatbot designed to help students,
future applicants, and academics learn more about Albukhary International University.

The chatbot provides quick and accurate information about university programs,
admission processes, scholarships, grading systems, faculties, and campus facilities.
It aims to improve user experience by offering instant support and guidance through
natural language conversations.

By using artificial intelligence, the system helps streamline information access,
supports better decision-making for students choosing majors, and enhances engagement
with the university in a fast and user-friendly way.

## Objectives
- Provide instant answers to university-related questions
- Help students choose majors and understand academic pathways
- Explain admission, scholarship, and grading criteria
- Introduce faculties, departments, and campus facilities
- Improve communication between the university and users

## Tech Stack
- Python
- Artificial Intelligence
- Machine Learning
- Natural Language Processing (NLP)
